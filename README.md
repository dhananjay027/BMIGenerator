# 🏋️ BMI Generator

A simple and responsive **Body Mass Index (BMI) Calculator** built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## 📌 About

This project calculates a user's BMI based on their height and weight and displays the result along with a health category (Underweight, Normal, Overweight, or Obese). It's a beginner-friendly frontend project demonstrating core web development skills.

---

## 🚀 Features

- Calculate BMI from height and weight inputs
- Instantly displays BMI value and health category
- Clean and user-friendly interface
- Fully client-side — no backend or server required
- Lightweight: pure HTML, CSS, and JavaScript

---

## 📊 BMI Categories

| BMI Range      | Category       |
|----------------|----------------|
| Less than 18.5 | Underweight    |
| 18.5 – 24.9    | Normal weight  |
| 25.0 – 29.9    | Overweight     |
| 30.0 and above | Obese          |

> BMI is calculated as: **BMI = weight (kg) / height² (m²)**
